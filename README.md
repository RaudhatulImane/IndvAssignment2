Brief Project Description:
This project is a Student Profile Management System developed using Java Servlet, JSP, JavaBean, JDBC, HTML, CSS, Bootstrap and Apache Derby Database. The system allows users to create, view, search and update student profiles.
List of Implemented Features:
•	Add Student Profile
•	Store Profile Information into Database
•	Display Profile Information
•	View All Profiles
•	Search Profile by Student ID
•	Search Profile by Name
•	Edit Existing Profile
•	Responsive User Interface using Bootstrap
